# Projects Portfolio

Hi, my name is **Alexander Tran**, and I am a student at **Iowa State University** studying Software Engineering with an interest in software development, AI/ML, backend programming, and IT-focused problem solving.

This repository contains a collection of projects I completed while building my foundation in programming. These projects helped me practice core computer science concepts such as object-oriented design, conditional logic, arrays, lists, inheritance, interfaces, abstraction, and testing.
